# 📊 NetSpeed Widget - Монитор скорости интернета

> "Когда твой интернет медленнее, чем загрузка этого виджета..." 🐌

---

## 📖 Описание / Description

**RU:** Оконный виджет мониторинга скорости интернета в реальном времени для Linux (особенно для Xfce4). Отображает вертикальный график загрузки (зелёный) и выгрузки (синий) с настраиваемыми размерами, цветами и прозрачностью. Показывает данные непосредственно из `/proc/net/dev`.

**EN:** A real-time internet speed monitoring widget for Linux (especially for Xfce4). Displays a vertical graph of download (green) and upload (blue) speeds with adjustable size, colors and transparency. Shows data directly from `/proc/net/dev`.

## 🎯 Особенности / Features

- 🚀 **Реальный мониторинг** - данные напрямую из `/proc/net/dev`
- 📊 **Вертикальный график** - два столбика: зелёный (download) и синий (upload)  
- 🎨 **Настраиваемость**:
  - Размер окна (растянуть за уголок "⇲")
  - Цвета всего и вся
  - Прозрачность 
  - Размер шрифта
  - Выбор сетевого интерфейса
- 🖱️ **Удобное управление**:
  - Перетаскивание левой кнопкой мыши
  - Правая кнопка = контекстное меню
- 💾 **Сохранение настроек** - всё запоминается между запусками
- 🔄 **Самообновление** - при сбросе настроек перезапускается автоматически

---

## 🛠 Совместимость / Compatibility

**RU:** 
- **Рекомендуется для Xfce4** (родной брат xfce4-netload-plugin)  
- Работает на любой X11-сессии Linux
- **Wayland не поддерживается** (прощай, будущее...)

**EN:**
- **Recommended for Xfce4** (native brother of xfce4-netload-plugin)
- Works on any X11 Linux session
- **Wayland not supported** (farewell, future...)

## 📦 Установка / Installation

1. **Скачайте `vilan.py`** в любую папку:
```bash
wget https://ваш_сервер/vilan.py -O vilan.py
```

2. **Сделайте исполняемым**:
```bash
chmod +x vilan.py
```

3. **Запустите**:
```bash
./vilan.py
# или для фона:
./vilan.py &
```

## 🧪 Настройка / Configuration

**Файл конфигурации:** `~/.netspeed_config`

Настройки сохраняются автоматически при каждом изменении:
```ini
[colors]
bg = #2d2d2d
download = #4CAF50
upload = #2196F3
grid = #444444
text = white

[size]
width = 200
height = 150

[font]
size = 9

[network]
interface = wlan0

[display]
transparency = 0.9
```
